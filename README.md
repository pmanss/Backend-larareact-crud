# Cara Pemasangan
<br>
1.jalankan composer install atau composer update <br>
2.copy .env.example dengan diganti namanya jadi .env<br>
3.setting db di .env<br>
4.jalankan php artisan migrate<br>
5.jalankan php artisan serve<br>
